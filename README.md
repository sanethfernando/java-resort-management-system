# Java Resort Management System

A console-based Java application that simulates a resort room
management system, built using Object-Oriented Programming principles.

## Features
- Room check-in and check-out
- Multiple room categories (Standard, Deluxe, Premium)
- Special services for accessibility-friendly rooms
- Exception handling with custom exceptions
- Persistent storage using file I/O
- Menu-driven CLI interface

## Technologies Used
- Java
- OOP (Inheritance, Abstraction, Interfaces, Polymorphism)
- File I/O
- Collections (ArrayList)

## How to Run
1. Clone the repository
2. Compile all `.java` files
3. Run `StageD.java`

## Purpose
This project was built to strengthen my understanding of Java OOP,
exception handling, and basic persistence mechanisms.
